## 🚀 Install

**We recommend using yarn or pnpm to install**，it not only makes development easier，but also allow you to take advantage of the rich ecosystem of Javascript packages and tooling.

```bash
$ yarn add pizza-ui
```

```bash
$ pnpm add pizza-ui
```
