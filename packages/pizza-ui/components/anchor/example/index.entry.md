# 侧边导航栏 Anchor

侧边导航栏，你懂的，顾名思义就是在旁边帮你找路的

## 演示

```example
basic.vue
fixed.vue
rail.vue
```

## API

### Anchor Props

| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| show-rail | `boolean` | `true` | 是否展示轨道 |  |
| smooth | `boolean` | `false` | 是否开始流畅滚动 |  |
| offsetTarget | `string | HTMLElement` | `document` | 滚动目标容器 |  |
